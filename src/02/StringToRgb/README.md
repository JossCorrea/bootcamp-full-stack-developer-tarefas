### StringToRgb

Dado uma entrada de dado de uma cor, por exemplo, vermelha, azul, amarela, verde, retorne o valor rgb correspondente:
vermelha: rgb(255,0,0)
azul: rgb(0,0,255)
amarela: rgb(255,255,0)
verde: rgb(0,128,0)
